7.
When an interface is configured with PortFast and BPDU guard, illustrate these features and
how the interface will respond when it received a BPDU?
When an interface is configured for PortFast and BPDU guard, BPDU guard will immediately
puts the switch port in an error-disabled state on receipt of any BPDU. It protects against
potential loops by effectively shutting down the port. Also, the BPDU guard feature provides
a secure response to invalid configurations because an administrator must manually put the
interface back into service.
With PortFast, it cause a layer 2 LAN interface configured as an access port to enter the
forwarding state immediately, by passing the listening and learning state. PortFast is used for
connecting end station to a layer 2 access port.

Answer

The alternate port and the backup port are the ports that are currently blocked. For a root port, the Alternate port is used as a backup port for the root port. Whenever the root port is unavailable, the alternate port is used to fill the void. The Backup port is a port that serves as a backup for a specific port. When the designated port expires, the backup port will become the new designated port, and the backup port will become the new designated port. When two spanning-tree devices' ports are connected to each other, the device creates a loop and blocks one of the ports. The backup port is the one that has been blocked and cannot be used.

 

To avoid loops, an alternate and backup port are both in a blocking (or discarding) state. Just on links in which neither end is a root port are alternate ports used. It is only one end of the segment that is blocked, whereas the other is still in forwarding mode, enabling a quicker switch to forwarding mode if needed.
